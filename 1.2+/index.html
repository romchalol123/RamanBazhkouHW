Редко использумые теги

Теги <dl>, <dt> и <dd>

Элементы <dl>, <dt>, <dd> предназначены для создания списка определений. Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина. Закрывающие теги </dt> </dd> не обязательны, поскольку следующий тег сообщает о завершении предыдущего элемента. Тем не менее, хорошим стилем является закрывать все теги.

Синтаксис

<dl>
<dt>Термин 1</dt>
<dd>Определение термина 1</dd>
<dt>Термин 2</dt>
<dd>Определение термина 2</dd>
</dl>

Отображение на странице

Термин 1
Определение термина 1
Термин 2
Определение термина 2

Тег <pre>

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один. Тег <pre> позволяет обойти эту особенность и отображать текст как требуется разработчику. В отличие от тега <plaintext>, использование которого осуждается в HTML 4, внутри контейнера <pre> допустимо применять любые теги кроме следующих: <big>, <img>, <object>, <small>, <sub> и <sup>. Закрывающий тег обязателен.

Синтаксис

<pre>Текст</pre>

Пример

<body>
<pre>

            |---||  |-----| |-----|
            |---||   |----  |-----| 
            |----|    ---   |-----|
            |-----          ------|
</pre>
</body>

Тег <code>

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.
В отличие от тега <pre> дополнительные пробелы внутри контейнера <code> не учитываются, так же, как и переносы текста. Поэтому используйте тег <br> или <p> для создания переносов.

Синтаксис

<code>Текст</code>

Пример

<body>
<p>Код программы</p>
<code>
function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}
</code>
</body>

Отображение на странице

Код программы

function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}


Тег <tt>

Контейнер <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования. Закрывающий тег обязателен.

Синтаксис

<tt>Текст</tt>

Пример

<body>
<p><tt>Programm code</tt></p>
<p>Programm code</p>
</body>

Отображение на странице

Programm code

Programm code
