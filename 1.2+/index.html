Теги <dl>, <dt> и <dd>

Элементы <dl>, <dt>, <dd> предназначены для создания списка определений. Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина. Закрывающие теги </dt> </dd> не обязательны, поскольку следующий тег сообщает о завершении предыдущего элемента. Тем не менее, хорошим стилем является закрывать все теги.

Синтаксис


<dl>
    <dt>Термин 1</dt>
        <dd>Определение термина 1</dd>
    <dt>Термин 2</dt>
        <dd>Определение термина 2</dd>
</dl>

Отображение на странице

Термин 1
Определение термина 1
Термин 2
Определение термина 2

Тег <pre>

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один. Тег <pre> позволяет обойти эту особенность и отображать текст как требуется разработчику. В отличие от тега <plaintext>, использование которого осуждается в HTML 4, внутри контейнера <pre> допустимо применять любые теги кроме следующих: <big>, <img>, <object>, <small>, <sub> и <sup>. Закрывающий тег обязателен.

Синтаксис

<pre>Текст</pre>

Тег <code>

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.
В отличие от тега <pre> дополнительные пробелы внутри контейнера <code> не учитываются, так же, как и переносы текста. Поэтому используйте тег <br> или <p> для создания переносов.

Синтаксис

<code>Текст</code>

Пример

<body> 
    <p>Код программы</p>
        <code>
            function checkParent (src, dest)
            {
                while (src != null)
                {
                    if (src.tagName == dest) return src
                    src = src.parentElement
                }
               return null
            }
        </code>              
</body>

Отображение на странице

Код программы
        function checkParent (src, dest)
        {
            while (src != null)
            {
                if (src.tagName == dest) return src
                src = src.parentElement
            }
            return null
        }

Тег <tt>

Контейнер <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования. Закрывающий тег обязателен.

Синтаксис

<tt>Текст</tt>

Пример

<body>
    <p><tt>Programm code</tt></p>
    <p>Programm code</p> 
</body>

Отображение на странице

Programm code

Programm code


Тег <kbd>

Тег <kbd> используется для обозначения текста, который набирается на клавиатуре или для названия клавиш. Браузеры обычно помечают текст в контейнере <kbd> моноширинным шрифтом. Закрывающий тег обязателен.

Синтаксис

<kbd>Текст</kbd>

Пример

<body>
    <p>
        Use the command <kbd>help mycommand</kbd> to view documentation
        for the command "mycommand".   
    </p>
</body>

Отображение на странице

Use the command help mycommand to view documentation for the command "mycommand".


Тег <var>

Тег <var> используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере <var> курсивным начертанием. Закрывающий тег обязателен.

Синтаксис

<var>Текст</var>

Пример

<body>
    <p>Переменная <var>a</var> должна быть равна переменной <var>select</var>,
    тогда выполнится следующий код: <code>Hello world !!!</code> </p>
</body>

Отображение на странице

Переменная a должна быть равна переменной select,
тогда выполнится следующий код: Hello world !!!


Теги <sup>, <sub>

<sup> ( <sub> ) отображают шрифт в виде верхнего (нижнего) индекса соответственно. Текст при этом отображается выше (ниже) базовой линии текста соответственно и уменьшенного размера.

Синтаксис

<sup>Текст</sup>
<sub>Текст</sub>

Пример

<body>
    <p>
        <i>X<sub>1</sub>+Y<sub>1</sub>=2<sup>2</sup></i>
    </p>
</body>

Отображение на странице

 X1+Y1=22

Теги <q>, <blockquote>

Тег <q> используется для выделения в тексте цитат. Содержимое контейнера автоматически отображается в браузере в кавычках.
Тег <blockquote> предназначен для выделения длинных цитат внутри документа. Текст, обозначенный этим тегом, традиционно отображается как выровненный блок с отступами слева и справа (примерно по 40 пикселов), а также с отбивкой сверху и снизу.

Синтаксис

<q>Текст</q>
<blockquote>Текст</blockquote>

Пример

<body>
    <p>Станислав Лец утверждал: <q>Чаще всего выход там, где был вход</q>.</p>

    <hr>      <!-- тег <hr> отображает горизонтальную линию, для наглядности -->
    <blockquote>
        <p>
            Lorem ipsum dolor sit amet, consectetuer adipiscing elit, 
            sed diem nonummy nibh euismod tincidunt ut lacreet dolore magna aliguam erat 
            volutpat. Ut wisis enim ad minim veniam, quis nostrud exerci tution ullamcorper 
            suscipit lobortis nisl ut aliquip ex ea commodo consequat.
        </p>
    </blockquote>
    <hr>
</body>

Отображение на странице

Станислав Лец утверждал: Чаще всего выход там, где был вход.

Теги <ins>, <del>

Тег <ins> предназначен для выделения текста, который был добавлен в новую версию документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <ins> как подчеркнутый.

Тег <del> используется для выделения текста, который был удален в новой версии документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <del> как перечеркнутый.

Синтаксис

<ins>Текст</ins>
<del>Текст</del>

Пример

<body> 
    <p>
        в последующих версиях <del>этот недостаток был устранен</del>
        <ins>подход разработчиков этого браузера был изменен</ins>.
    </p>    
</body>

Отображение на странице

в последующих версиях этот недостаток был устранен подход разработчиков этого браузера был изменен.


Тег <adress>

Тег <adress> предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д. Планируется, что поисковые системы будут анализировать содержимое этого тега для сбора информации об авторах сайтов.
По умолчанию текст внутри контейнера <adress> отображается курсивным начертанием. Если эта особенность не требуется, используйте стили для изменения начертания шрифта.

Синтаксис

<adress>Текст</adress>

Пример

</body>
    <p>
        Официальный сайт Юникода располагается по адресу www.unicode.org.<br>
        <adress>Официальный сайт Юникода располагается по адресу www.unicode.org.</adress>
    </p>
</body>

Отображение на странице

Официальный сайт Юникода располагается по адресу www.unicode.org.

Официальный сайт Юникода располагается по адресу www.unicode.org.


Теги <nobr>, <wbr>

Тег <nobr> уведомляет браузер отображать текст без переносов. Если этого тега в коде документа нет, а также имеются переводы строки, они игнорируются и текст выравнивается по левому краю окна браузера или родительского элемента. При этом браузер переводы строк расставляет автоматически, чтобы текст полностью поместился по ширине окна. Использование тега <nobr> заставляет отображать текст без переносов, одной строкой, что может привести к появлению горизонтальной полосы прокрутки. Пользователям придется прокручивать текст по горизонтали, чтобы увидеть его полностью, чего обычно делать никто не любит.
Тег <wbr> указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента.

Синтаксис

<nobr>Текст</nobr>

Текст<wbr>Текст

Пример

<body>
    <p><nobr>Lorem ipsum dolor sit amet, consectetuer adipiscing elit, 
    sed diem nonummy nibh euismod tincidunt ut lacreet dolore 
    magna aliguam erat volutpat. Ut wisis enim ad minim veniam, 
    quis nostrud exerci tution ullamcorper suscipit lobortis nisl 
    ut aliquip ex ea commodo consequat.</nobr></p>

    <p class="word">метоксихлор<wbr>диэтиламино<wbr>метилбутил<wbr>аминоакридин_<wbr>метоксихлор<wbr>диэтиламино<wbr>
    метилбутил<wbr>аминоакридин_<wbr>метоксихлор<wbr>диэтиламино<wbr>метилбутил<wbr>аминоакридин</p>
</body>
